class lib.density.integrate.importanceSampling.ImportanceSampleIntegrateNormal(f, p, ranges=None)[source]¶

Bases: object

Importance Sampling is the idea that we want to solve the integral of the form:

\[\mathbb E(f) = \int_{-\infty}^{\infty} f(\mathbf w) p(\mathbf w) d \mathbf w\]

We shall sample directly using the multivariate Gaussian distribution that we know, rather than using uniform sampling. That should significantly reduce the amount of time required. However for this we shall need to change the equations a bit.

\begin{align*} \mathbb E(f) &= \int_{-\infty}^{\infty} f(\mathbf w) p(\mathbf w) \frac {q(\mathbf w)} {q(\mathbf w)} d \mathbf w \\ &= \int_{-\infty}^{\infty} \Big( f(\mathbf w) \frac {p(\mathbf w)} {q(\mathbf w)} \Big) q(\mathbf w) d \mathbf w \\ &= \frac 1 N \sum_{i=1}^N w_s{f(w_i)}, w_i \sim q(\mathbf w) \end{align*}

The important thing about this transformation is that the sampling form another distribution (like \(q(\mathbf w)\)) is typically easier to do.

For unnormalized distributions, we can use the following formula instead:

\begin{align*} \mathbb E(f) &= \frac 1 {\sum_{i=1}^N w_s} \sum_{i=1}^N w_s{f(w_i)}, w_i \sim q(\mathbf w) \end{align*}

initialize the Importance Sampler.

Parameters

  • f (function) – This takes a numpy nd-array of dimensions \((N,d)\) representing a set of \(N\) vectors of dimension \(d\) and return \(N\) values that needs to be integrated over the \(d\) dimensional vector space.

  • p (function) – This takes a numpy nd-array of dimensions \((N,d)\) representing a set of \(N\) vectors of dimension \(d\) and return \(N\) values that represents the probability density function over the \(d\) dimensional space.

  • ranges (list of lists, optional) – This is a list of \(d\) ranges, one for each dimension over which the \(d\) dimensional vector space is going to be uniformly sampled, by default None

class lib.density.integrate.importanceSampling.ImportanceSampleIntegrateNormalAdaptive(f, p, ranges=None)[source]¶

Importance Sampling is the idea that we want to solve the integral of the form:

\[\mathbb E(f) = \int_{-\infty}^{\infty} f(\mathbf w) p(\mathbf w) d \mathbf w\]

Rather than uniform sampling, it is possible that we use a Gaussian distribution to sample form. This can be easily accomplished if we use a Gaussian distribution \(q\) that closely represents our required distribution \(p\). This can be donee by finding the parameters of a multivariate Gaussian distribution that closely matches the required distribution using the KL-divergence.

class lib.density.integrate.importanceSampling.ImportanceSampleIntegrateUniform(f, p, ranges=None)[source]¶

Bases: object

Importance Sampling is the idea that we want to solve the integral of the form:

\[\mathbb E(f) = \int_{-\infty}^{\infty} f(\mathbf w) p(\mathbf w) d \mathbf w\]

It is entirely possible that one samples from the distribution \(p(\mathbf w)\). Under this formulation, the equation for importance sampling (in the limit of infinite samples \(N\)) would be given by the following:

\[\mathbb E(f) = \int_{-\infty}^{\infty} f(\mathbf w) p(\mathbf w) d \mathbf w\]
\[\mathbb E(f) = \frac 1 N \sum_{i=1}^N {f(w_i)}, w_i \sim p(\mathbf w)\]
